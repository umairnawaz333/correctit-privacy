CorrectIt - Privacy Policy

Last updated: April 5, 2026

Overview

CorrectIt is a Chrome extension that helps users correct and improve their text using AI. We are committed to protecting your privacy and being transparent about how your data is handled.

Data Collection

CorrectIt does not collect, store, or transmit any personal data to our servers. We do not have servers — the extension runs entirely in your browser.

How Your Data Is Used

  • Text you correct: When you use CorrectIt, the text you select or type is sent directly from your browser to the AI provider you have configured (Google Gemini or OpenAI). This communication happens directly between your browser and the provider's API — we never see or store this data.
  • API Key: Your API key is stored locally in your browser using Chrome's storage API. It is never transmitted to anyone other than the AI provider you configured.
  • Settings: Your preferences (provider, model selection) are stored locally in your browser and are never transmitted externally.

Third-Party Services

CorrectIt communicates with third-party AI providers based on your configuration:

  • Google Gemini API — subject to Google's API Terms of Service
  • OpenAI API — subject to OpenAI's Terms of Use

Please review the respective privacy policies of these providers to understand how they handle data sent via their APIs.

Permissions

  • activeTab: Used to access the currently active tab so the extension can read selected text and replace it with corrected text.
  • storage: Used to save your API key and preferences locally in your browser.
  • contextMenus: Used to add right-click menu options for quick text correction.

Data Security

All data stays in your browser. Your API key is stored using Chrome's built-in storage API, which is sandboxed to the extension.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
