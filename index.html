CorrectIt - Privacy Policy

Last updated: April 5, 2026

Overview

CorrectIt is a Chrome extension that helps users correct and improve their text using AI. We are committed to protecting your privacy and being transparent about how your data is handled.

Data Collection

CorrectIt does not collect, store, or transmit any personal data to our servers. We do not have servers — the extension runs entirely in your browser.

How Your Data Is Used

  • Text you correct: When you use CorrectIt, the text you select or type is sent directly from your browser to the AI provider for processing. By default, the extension uses OpenRouter to access free AI models. This communication happens directly between your browser and the provider's API — we never see or store this data.
  • API Key (optional): If you choose to use your own API key (e.g. Groq), it is stored locally in your browser using Chrome's storage API. It is never transmitted to anyone other than the AI provider you configured.
  • Settings: Your preferences (provider, model selection) are stored locally in your browser and are never transmitted externally.

Third-Party Services

CorrectIt communicates with third-party AI providers to process text corrections:

  • OpenRouter (default) — routes requests to free AI models. Subject to OpenRouter's Terms of Service
  • Groq API (optional) — subject to Groq's Terms of Use

Please review the respective privacy policies of these providers to understand how they handle data sent via their APIs.

Permissions

  • activeTab: Used to access the currently active tab so the extension can read selected text and replace it with corrected text.
  • storage: Used to save user preferences locally in your browser.
  • contextMenus: Used to add right-click menu options for quick text correction.
  • Content script (all URLs): Required to display the floating correction toolbar when you select text on any webpage.

Data Security

All data stays in your browser. Any optional API keys are stored using Chrome's built-in storage API, which is sandboxed to the extension. No data is sent to any server owned or operated by CorrectIt.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.

Contact
